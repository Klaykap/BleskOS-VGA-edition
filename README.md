# BleskOS-VGA-edition
Part of BleskOS operating system project for oldest computers

## Version Alpha 2
This version contains these drivers:
* VGA
* PC speaker
* Keyboard
* PS/2 mouse
* Easy filesystem for USB and hard disk

And contains these programs:
* Text editor
* Graphic editor
* Sound editor
* Calculator
